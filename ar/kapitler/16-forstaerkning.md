# الفصل 16 — التضخيم

**اللغة:** [🇩🇰 Dansk](../../da/kapitler/16-forstaerkning.md) · [🇬🇧 English](../../en/kapitler/16-forstaerkning.md) · **🇸🇦 العربية**

## 🎯 أهداف التعلم

- فهم مفهوم التضخيم ومعامل الكسب
- معرفة توصيلات الترانزستور الأساسية للمكبّر
- فهم توصيلات الباعث المشترك، المجمّع المشترك، القاعدة المشتركة
- تطبيق مكبّرات DC في أنظمة التحكم
- نظرة عامة على مكبّر العمليات (op-amp)

---

## 16.1 ما هو المكبّر؟

**المكبّر** يزيد من سعة أو قدرة الإشارة.

### معامل الكسب

ثلاثة أشكال:
- **كسب الجهد**: $A_U = \dfrac{U_{out}}{U_{in}}$
- **كسب التيار**: $A_I = \dfrac{I_{out}}{I_{in}}$
- **كسب القدرة**: $A_P = \dfrac{P_{out}}{P_{in}}$

### الديسيبل (dB)
يُعبَّر عن الكسب كثيراً بالديسيبل (مقياس لوغاريتمي):
$$A[dB] = 20 \cdot \log\left(\frac{U_{out}}{U_{in}}\right)$$

| المعامل | dB |
|---------|----|
| 2 | 6 dB |
| 10 | 20 dB |
| 100 | 40 dB |
| 1,000 | 60 dB |

> 💡 +3 dB = مضاعفة **القدرة**، +6 dB = مضاعفة **الجهد/التيار**.

## 16.2 مكبّرات DC

في أنظمة التحكم تُستخدم **مكبّرات DC** في مواضع كثيرة:

| التطبيق | الوظيفة |
|---------|---------|
| **مكبّر الحسّاس** | تكييف إشارة الحسّاس للمنطق |
| **دارة قيادة** | تضخيم إشارة المنطق لتشغيل مرحّل/محرّك |
| **مكبّر القياس** | تضخيم جهود قياس صغيرة |
| **مكبّر التحكم** | في وحدات تحكم PID |

### مثال — دارة قيادة

تُعطي دارة المنطق 12 V و 1 mA. لتنشيط مرحّل بحاجة لـ 50 mA، يُستخدم ترانزستور كمفتاح:

```
   +12 V
      │
      [مرحّل]
      │
      ─┤├─ ← ديود (ديود حماية ضد ذُروات الحثّ)
      │
   ───┤  ← C
   منطق──[R₁]──┤  ← B (NPN)
       (12 V) ───┤
                 ↓ E
                ───
                 ⏚
```

مع كسب تيار $h_{FE} = 200$:
$$I_B = I_C / h_{FE} = 50/200 = 0.25 \text{ mA}$$

يستطيع المنطق توفير 0.25 mA بسهولة → الدارة تعمل.

## 16.3 توصيلات الترانزستور

هناك 3 توصيلات أساسية لـ BJT، تُعرَّف بالطرف **المشترك** بين الدخل والخرج:

### الباعث المشترك (CE)
**التوصيل الأكثر استخداماً للمكبّرات**.

| الخاصية | القيمة |
|---------|--------|
| كسب الجهد | عالٍ (10–1000) |
| كسب التيار | عالٍ (β) |
| ممانعة الدخل | متوسطة (~kΩ) |
| ممانعة الخرج | عالية (~10 kΩ) |
| إزاحة الطور دخل→خرج | **180°** |

### المجمّع المشترك (CC — تابع الباعث)
| الخاصية | القيمة |
|---------|--------|
| كسب الجهد | ≈1 (لا تضخيم!) |
| كسب التيار | عالٍ |
| ممانعة الدخل | **عالية جدّاً** |
| ممانعة الخرج | **منخفضة جدّاً** |
| إزاحة الطور | 0° |

> 💡 يُستخدم كـ **عازل (buffer)** أو لمطابقة الممانعة — مثلاً بين حسّاس عالي الممانعة وحِمل منخفض الممانعة.

### القاعدة المشتركة (CB)
| الخاصية | القيمة |
|---------|--------|
| كسب الجهد | عالٍ |
| كسب التيار | ≈1 |
| ممانعة الدخل | **منخفضة جدّاً** |
| ممانعة الخرج | عالية جدّاً |
| إزاحة الطور | 0° |

> 📡 يُستخدم في دارات RF والتضخيم بترددات عالية.

## 16.4 نقطة العمل

لتضخيم إشارة **AC** بدون تشويه، يجب أن يكون الترانزستور مُحيَّزاً عند **نقطة عمل** في منتصف منطقته الخطية.

```
   I_C
    │
    │     /  <- إشباع
    │    /
    │   X     <- نقطة العمل
    │  /
    │ /
    │/
    └──────────  V_CE
       ↑
     قطع
```

مقاومات التحيّز على القاعدة تضع تياراً ثابتاً صغيراً يحدّد نقطة العمل بصورة صحيحة.

## 16.5 مكبّر العمليات (op-amp)

**op-amp** مكبّر DC عالي الكسب في صورة دائرة متكاملة. **أهمّ مكوّن تماثلي** في الإلكترونيات الحديثة بعد MOSFET.

### الخصائص (المثالية)
- **الكسب**: عالٍ جدّاً (~10⁵ – 10⁶)
- **ممانعة الدخل**: عالية جدّاً (لا نهائية)
- **ممانعة الخرج**: منخفضة جدّاً (0)
- مدخلان: **+** (غير عاكس) و **−** (عاكس)
- مخرج واحد

### الرمز
```
        ┌───────┐
   + ───┤        \
        │         ├── خرج
   - ───┤        /
        └───────┘
```

### القاعدة الذهبية (مع التغذية العكسية السالبة)
1. يتغيّر الخرج حتى يصبح المدخلان عند **نفس الجهد**
2. **لا يدخل تيار** إلى المدخلين

## 16.6 توصيلات op-amp الشائعة

### المكبّر العاكس

```
        Rf
   ┌────[▭]────┐
   │           │
   ├──[▭]──┤-  │
   U_in    │  ├─── U_out
   ┌──────┤+  │
   │       └───┘
   ⏚
```

**الكسب**:
$$A_U = -\frac{R_f}{R_{in}}$$

(الإشارة السالبة تعني عكس الطور)

### المكبّر غير العاكس

```
   U_in ──┤+
          │ ├─── U_out
       ┌──┤-
       │  │
   ┌───┴─[Rf]─── خرج
   │
  [R₁]
   │
   ⏚
```

**الكسب**:
$$A_U = 1 + \frac{R_f}{R_1}$$

### تابع الجهد (buffer)
حالة خاصّة من غير العاكس: $A_U = 1$.
يُستخدم لـ **عزل الممانعة** بدون تضخيم.

### المكبّر الجامع
يجمع عدّة مدخلات بأوزان.

### المكبّر التفاضلي
يضخّم الفرق بين إشارتين — يُستخدم لمكبّرات القياس وتكييف إشارات الحسّاسات.

### المقارن (Comparator)
يقارن جهدَين — الخرج ON أو OFF (رقمي):
- إذا $U_+ > U_-$ → الخرج HIGH
- إذا $U_+ < U_-$ → الخرج LOW

> 🔍 يُستخدم في الإنذارات، كاشفات المستوى، محوّلات A/D.

## 16.8 المكبّر الدافع-الجاذب (push-pull) / الفئة AB

ترانزستوران (NPN + PNP) يتقاسمان العمل:
- NPN يتعامل مع النصف الموجب من الإشارة
- PNP يتعامل مع النصف السالب

> 🔊 يُستخدم في مكبّرات الصوت — كفاءة أفضل من الفئة A. الفئة AB هي الخيار القياسي في الـ Hi-Fi.

| الفئة | الكفاءة | التشويه |
|-------|---------|---------|
| A | 25% | منخفض |
| AB | 50–60% | منخفض (مع تشويه عبور) |
| B | 78% | أعلى |
| D (تبديل) | 90+% | منخفض جدّاً (حديث) |

## 📌 ملخّص

- **التضخيم** = زيادة سعة الإشارة. يُحسب كنسبة (Aᵤ = U_خرج / U_دخل) أو بالديسيبل.
- **3 توصيلات أساسية للترانزستور**:
  - **CE (الباعث المشترك)**: التضخيم الأشهر، انعكاس 180°.
  - **CC (المجمّع المشترك / تابع الباعث)**: تضخيم ~1، ممانعة دخل عالية، خرج منخفضة — يُستخدم كمحوّل ممانعة.
  - **CB (القاعدة المشتركة)**: للترددات العالية.
- **مكبّر العمليات (op-amp)**: مكبّر متكامل بتضخيم هائل، ممانعة دخل عالية جدّاً.
- **توصيلات op-amp الأساسية**: عاكس، غير عاكس، تابع جهد، جامع، تفاضلي، مقارن.
- **استخدام في PLC**: مكبّر إشارة من الحسّاس قبل دخوله إلى المنطق الرقمي.
