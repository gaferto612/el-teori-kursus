| Tool | What it does |

## 🎯 How to use the course

1. **Read a chapter** — start with the summary (typically 1-2 pages).
2. **Open the interactive tool** for the topic, if there is one.
3. **Solve the exercises** without looking — check afterwards.
4. **Take the quiz** once you've gone through several chapters.

> 💡 **Tip**: If you're aiming for a specific exam (e.g. electrical installer, automation, boiler operator), focus on chapters 9–13 (electrical theory core) and 21–22 (measurement technology).
