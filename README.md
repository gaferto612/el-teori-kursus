## 📚 What's inside / Hvad indeholder kurset / ماذا يحتوي

- **22 chapters** covering math fundamentals, natural sciences, DC/AC theory, components, electronics, and measurement
- **22 exercise sets** with solutions
- **6 interactive browser tools** (Ohm's law, AC visualizer, logic gates, quiz, etc.)
