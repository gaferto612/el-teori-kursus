# ⚡ El-Teori Kursus

![License](https://img.shields.io/badge/license-MIT-blue) ![Kapitler](https://img.shields.io/badge/kapitler-22-green) ![Værktøjer](https://img.shields.io/badge/interaktive%20værktøjer-6-orange) ![Sprog](https://img.shields.io/badge/sprog-dansk%20%2B%20%D8%A7%D9%84%D8%B9%D8%B1%D8%A8%D9%8A%D8%A9-lightgrey)

Selvstudie-kursus baseret på lærebogen **El-Teori** (EFU, Den Jydske Haandværkerskole, 2006).
Materialet består af **resuméer + opgaver + interaktive øvelser** — på dansk med arabiske forklaringer ved siden af de tekniske termer.

> دورة في النظرية الكهربائية — ملخصات وتمارين وأدوات تفاعلية، باللغة الدنماركية مع شروحات عربية للمصطلحات الفنية.

## 🎯 Sådan bruger du kurset

1. **Læs et kapitel** — start med resuméet (typisk 1-2 sider).
2. **Åbn det interaktive værktøj** for emnet, hvis der findes ét.
3. **Løs opgaverne** uden at kigge — tjek bagefter.
4. **Tag quizzen**, når du har været igennem flere kapitler.

> 💡 **Tip**: Hvis du går efter en konkret eksamen (fx el-installatør, automation, kedelpasser), så fokusér på kap. 9–13 (el-teori-kerne) og kap. 21–22 (måleteknik).
